CS 2350: CLIENT-SIDE WEB DEVELOPMENT
This course provides an introduction to client-side programming and Web page development. Subjects covered include responsive Web page design and dynamic Web page development. The course will explore various technologies such as HTML5, CSS3, and Javascript.
Class project - 150p
Cs2350 Class project-Index
In this project, you will be creating a website of your choosing. This project will need to meet the following high-level requirements:
  • It must use bootstrap.
  • It must consist of at least five HTML pages.
  • The landing page (index.html) must have a top navbar, a jumbotron with a background image, featured content, a footer.
  • All pages must be responsive mobile-first with support for small, medium, and large devices (use bootstrap sm, md, and lg breakpoints).
  • It must use the following JavaScript features: functions, objects, arrays, events,
  • It must use multimedia (images or videos).
